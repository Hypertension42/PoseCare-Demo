# PoseCare Demo

医学健康方向的赛道二 Demo：用视觉和语音理解用户当前姿势，输出姿势风险、问题标注、纠正建议和可分享报告卡。

## 产品形态

主形态：移动端 Web/H5。

内部能力：AI 姿势分析 Agent。

传播出口：姿势体检卡。

这三个关系不要混淆：

- Web/H5 是产品外壳，负责相机、上传、分析结果和展示。
- Agent 是内部分析脑，负责把姿态指标、用户问题和健康建议组织成自然回答。
- 报告卡是结果产物，用来摆摊展示、分享传播和增强比赛记忆点。

## MVP 主流程

1. 用户打开网页，进入相机/上传页。
2. 选择场景：坐姿、站姿、深蹲、康复动作。
3. 拍一张照片或上传一段短视频。
4. 用户语音或输入问题，例如“我这个坐姿哪里不对？”
5. 系统识别人体关键点，叠加骨架和风险标注。
6. 系统输出风险等级、问题解释和纠正建议。
7. 生成一张“姿势体检卡”。
